# create-dmg-action

This is specifically for TazUO Launcher

## Inputs

## `name`

**Required** Dmg file name. Default `"target"`.

## `srcdir`

**Required** Source dir path. Default `"srcdir"`.

## Example usage

```
uses: QQxiaoming/create-dmg-action@v0.0.2
with:
  name: 'target'
  srcdir: 'srcdir'
```
